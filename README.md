# 常见风险信号

- 先交保证金才能开始合作
- 保证金不写退款条件
- 承诺保底销量
- 承诺快速起店
- 承诺稳赚不赔
- 不断要求升级套餐
- 不断要求追加广告费
- 不断要求充值
- 所有承诺只存在于聊天记录
